-객체(object) : 어떠한 대상의 속성(정보/상태 등) 정의 하는 것
    -{ } 중괄호 이용한 자료의 타입
    -{ 속성명 : 속성값, '속성명' : 속성값 }
    - 객체는 주로 변수/상수 에 저장/참조 , let/const 변수 = { }
    -.(접근/도트/참조)연산자 통해 객체내 속성 값 호출, 변수.속성명
    - object.keys( 객체 ) : 모든 속성명 반환 , object.value( 객체 ) : 모든 속성값
    - 객체 내 속성 추가/수정 : 변수.속성명 = 새로운 값
    - 중첩 : {key : [ ] , key : { } , key : function(){} }, 속성값에는 다양한 타입에 대입된다.
